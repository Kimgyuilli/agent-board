# Create Your First Phase

Phases are columns on your kanban board that organize tasks by workflow stage.

## Using MCP Tools

Ask your AI agent to create a phase:

```
Use the add_phase tool to create a phase called "Backlog"
```

Or use the `batch` tool to create multiple phases at once:

```
Use the batch tool to create phases: "Backlog", "In Progress", "Done"
```

## Next Steps

Once you have phases, you can:
1. **Add tasks** with `add_task` — assign them to phases
2. **Claim tasks** with `claim` — assign an agent to work on them
3. **Track progress** — watch the board update in real time!
