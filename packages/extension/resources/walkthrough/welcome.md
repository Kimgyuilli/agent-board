# Welcome to Agent Board

Agent Board is a VS Code extension that helps you manage AI agent development tasks with a real-time kanban board.

## What you can do

- **Organize tasks** into phases on a visual kanban board
- **Track agent activity** with real-time progress logs
- **Drag and drop** tasks between phases
- **Get notified** when agents complete or get blocked on tasks

Agent Board works with any MCP-compatible AI agent, giving you full visibility into your AI-assisted development workflow.
