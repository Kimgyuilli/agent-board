# Open the Board

There are two ways to open Agent Board:

## Option 1: Command Palette
1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
2. Type **"Agent Board: Show Board"**
3. Press Enter

## Option 2: Activity Bar
Click the **Agent Board** icon in the Activity Bar (left sidebar) to open the board view.

The board will display your phases and tasks in a kanban layout. If this is your first time, the board will be empty — follow the next steps to set it up!
